Instructions

1. Click start to start playing game.
2. Please Enter you name in the dialog box when you click on start.
3. Once you have entered your, the time start count.
4. Select any Object by clicking it from the object section.
5. Base on selection as compared to computer's choice you can get the a point.
6. The first person (You or Computer) to score 5 points wins the game.
